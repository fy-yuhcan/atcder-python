# 問題番号(カテゴリ)
217C - Inverse of Permutation (逆置換)
# 方針
Nは長さNの順列
Pは12Nで与えられている順列
Qはp_i番目の要素が i である順列

Q[P[i]] = i


## カテゴリ
- 逆置換
- 順列

## 詰まったところ
Q[P[i]] = iを思い付かなかった
ゆっくり追うことで、思いつけた
- P は「位置 → 値」を表す順列
- Q は「値 → 位置」を表す順列


## 次に出たら
q_{p_i} = i みたいな条件」が出てきたら、まず
i を 1,2,3 に具体的に代入して日本語に言い換える
置→値」と「値→位置」の 対応表を書く：
行1：位置 i
行2：値 P[i]
```python
# 逆置換を作る定番形
for i in range(1, N + 1):
    Q[P[i]] = i
```
